A powerful all-in-one WordPress plugin with 5 unique tools to convert YouTube content into timestamped links, notes, blog articles, quote images, and multi-platform formats.

== Description ==

🎯 **YouTubeConverter** is the ultimate WordPress plugin for creators, marketers, educators, and students who want to **repurpose YouTube content** into multiple formats – fast and effortlessly!

With this plugin, you get **5 tools in one**:

1. **🎬 Timestamp Clipper:** Generate shareable timestamped YouTube URLs.
2. **📝 Subtitle to Notes Generator:** Convert raw subtitles into clean bullet-point notes.
3. **📰 Video to Blog Generator:** Simulate a full blog article from a transcript.
4. **🖼️ Quote Image Generator:** Turn quotes into beautiful shareable images.
5. **🔀 Multi-Platform Converter Toolkit:** Get optimized content ideas for Shorts, Reels, Podcasts, Tweets, and more.

No APIs, no backend, no logins. Just paste and convert.

== Features ==

- Clean and modern responsive UI
- Works with shortcodes – embed tools anywhere
- Dark theme layout for professional feel
- No API keys or technical setup needed
- Perfect for YouTubers, course creators, journalists, and digital marketers

== Installation ==

1. Upload the plugin folder `YouTubeConverter` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcodes listed below to add tools to your pages/posts.

== Shortcodes ==

Use these shortcodes to add tools wherever you like:

- `[yt_timestamp_clipper]` – YouTube Timestamp Clipper
- `[yt_subtitle_notes]` – Subtitle to Notes Generator
- `[yt_video_blog]` – YouTube to Blog Generator
- `[yt_quote_image]` – Quote Image Generator
- `[yt_multi_toolkit]` – Multi-Platform Converter Toolkit

== Frequently Asked Questions ==

= Does this plugin download YouTube videos? =
No. This plugin is strictly for **content repurposing** and **SEO-friendly productivity use**, not downloading or ripping.

= Can I customize the styles? =
Yes! You can modify the `style.css` file inside the plugin folder.

= Does it require YouTube API? =
No. All tools are frontend only — no API, backend, or external service is used.
